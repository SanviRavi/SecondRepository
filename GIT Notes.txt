CLONE:

First time copying from git repository to local repository.
Git Clone is used only once, when we are connecting to the Git repository for the first time from our local repository.
git clone https://github.com/SanviRavi/SecondRepository.git


PULL:
Updating the local repository

.git - Is a local git repository.

git add test.txt - add file to staging area

git commit - commit the newly added or changed files from staging area to local git repository
          git commit -m "add message"

git push - commit changes from local repository to git repository
           git push origin main

git status - track the added/modified files in local git repository 

git init - initialse the empty repository

git branch 'branchname' - create the branch

git checkout 'branchname' - connect to the branch

git branch -m prepod prod  -- to rename the branch names
